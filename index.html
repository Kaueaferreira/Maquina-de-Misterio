    Sobre o Projeto

    Objetivo

    O objetivo principal deste projeto é explorar conceitos de robótica e automação, utilizando a plataforma Arduino para construir um carrinho capaz de navegar sozinho, detectar obstáculos e realizar ações de acordo com comandos programados.

    Componentes

    • Placa Arduino Uno: Controla o funcionamento do carrinho e a comunicação com os sensores.
    • Motores DC: Responsáveis pelo movimento do carrinho.
    • Sensores de distância (ultrassônicos): Detectam obstáculos à frente, permitindo que o carrinho mude de direção.
    • Estrutura do carrinho: Feita para ser leve e resistente, permitindo manobras ágeis.

    Funcionamento do Carrinho

    A principal tarefa do carrinho é navegar sozinho, evitando obstáculos enquanto se move. Para isso, o Arduino será responsável por controlar os sensores (para detectar obstáculos) e os motores (para movimentar o carrinho).

    Sensores de Distância (Ultrassônicos):

    • O Arduino usa sensores ultrassônicos para medir a distância até os objetos à frente do carrinho.
    • Esses sensores emitem ondas sonoras e, ao perceberem o retorno dessas ondas, calculam a distância até o objeto mais próximo.
    • O sensor envia essa informação de distância para o Arduino.

    Analisando as informações dos Sensores:

    • O Arduino constantemente lê os valores dos sensores ultrassônicos.
    • Se o sensor detecta que um obstáculo está muito perto (por exemplo, a menos de 10 cm), isso significa que o carrinho está prestes a colidir com algo.
    • A partir daí, o Arduino toma uma decisão: parar, retroceder ou desviar.

    Controle dos Motores:

    • O Arduino envia comandos para os motores DC para movimentar o carrinho conforme a necessidade.
    • Se não houver obstáculos: o Arduino envia um comando para os motores avançarem e o carrinho continua sua trajetória.
    • Se houver um obstáculo muito perto: o Arduino pode emitir comandos para os motores pararem ou retrocederem.
    • Se houver obstáculos de lado: o Arduino pode acionar os motores para girar o carrinho e desviar do obstáculo.

    Lógica de Decisão:

    A lógica que faz o carrinho tomar essas decisões é programada no código do Arduino. A programação pode ser feita de forma simples, usando condições (if/else) e loops para fazer o carrinho verificar constantemente os sensores e executar as ações.

    Comandos do Carrinho:

    • Avanço: Para mover o carrinho para frente.
    • Retrocesso: Para fazer o carrinho dar ré, evitando o obstáculo.
    • Giro: Para desviar, girando o carrinho para a esquerda ou direita.
